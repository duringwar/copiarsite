# About

Ferramenta simples feita em Python para copiar estruturas públicas de sites autorizados.

O script:

- Lê `robots.txt`
- Procura `sitemap.xml`
- Coleta páginas do domínio
- Baixa HTML, CSS, JS e imagens
- Mantém estrutura original
- Ajusta links locais
- Funciona offline
